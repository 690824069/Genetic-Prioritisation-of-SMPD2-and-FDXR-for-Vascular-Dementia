# Upload Guide for GitHub and Zenodo, Version 6.7

Prepared: 2026-06-14

## GitHub Upload

Recommended upload folder or zip:

- `GitHub_Release_6.7`
- `GitHub_Release_6.7_20260614.zip`

Purpose:

- Public reproducibility repository
- Code
- Derived small data
- Supplementary tables S1-S5
- Figure source data
- Data availability and third-party provenance documentation

Do not upload to GitHub:

- Full FinnGen GWAS files
- Full eQTLGen files
- Full GTEx archives
- 1000 Genomes PLINK reference files
- Raw third-party GEO/CELLxGENE downloads unless licensing has been checked
- Cover letter or journal-submission-only correspondence

## Zenodo Upload

Recommended upload folder or zip:

- `Zenodo_Archive_6.7`
- `Zenodo_Archive_6.7_20260614.zip`

Purpose:

- Frozen DOI archive of the release
- Supplementary tables
- Figure source data
- Processed manuscript-level outputs
- Scripts
- Reproducibility notes
- Zenodo metadata draft
- Manuscript reference copy, if journal policy allows public or embargoed deposition

Suggested Zenodo upload type:

- Dataset

Suggested license:

- CC-BY-4.0 for derived tables, documentation, and figure source data
- MIT for scripts

If journal policy requires delayed release, use Zenodo embargo until publication.
