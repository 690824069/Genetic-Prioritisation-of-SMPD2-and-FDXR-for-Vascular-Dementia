# Data Availability

This repository contains derived manuscript outputs, processed source data for figures, supplementary tables, and audit files. The following original data sources are not redistributed and should be accessed from the original providers:

| Resource | Use in study | Redistribution status |
|---|---|---|
| eQTLGen whole-blood cis-eQTL summary statistics | Blood cis-eQTL instruments and post hoc instrument-resolution audit | Not redistributed; obtain from eQTLGen |
| FinnGen R11 vascular dementia GWAS | Primary vascular dementia outcome | Not redistributed; obtain from FinnGen according to FinnGen terms |
| GTEx v8 Brain Frontal Cortex BA9 eQTL data | Brain BA9 cross-tissue assessment and independent eQTL audit | Not redistributed; obtain from GTEx portal |
| 1000 Genomes European reference panel | LD clumping and pairwise LD calculations | Not redistributed; obtain from 1000 Genomes/approved mirrors |
| GEO datasets GSE186798, GSE22255, GSE58294 | External transcriptomic contextualisation | Raw/processed public data should be obtained from GEO |
| CZ CELLxGENE Human Brain Cell Atlas v1.0 | Single-cell cellular context | Access from CZ CELLxGENE |

## Included Derived Data

Included files are limited to small derived tables, figure source data, supplementary tables, locus-level summary files generated for visualisation/audit, and analysis logs or scripts. These files are intended to support reproducibility and transparency of the reported manuscript results.

## Recommended Public Archive

For journal submission, upload the GitHub repository for active code review and deposit the frozen release archive in Zenodo to obtain a DOI. The Zenodo DOI should then be cited in the manuscript data availability statement.
